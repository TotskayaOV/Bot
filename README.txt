Общие команды:
/start - в ответ на команду бот присылает id телеграмма пользователя, для передачи его администратору и присвоения роли.

Команды администраторов:
/del - удаление пользователя. Для удаления необходимо знать телеграмм id пользователя. В БД фигурирует в столбце user_id
/add - добавление пользователя. Добавляются поля:
    1. id - присваивается автоматически внутри БД
    2. user_id - телеграмм id пользователя (получается пользователем при первом обращении к боту через команду /start)
    3. name - фамилия пользователя
    4. user_role - в зависимости от указанной роли пользователя доступны разные функции бота и получения разных
    информационных сообщений о статусах агентов в гугл-таблицах
